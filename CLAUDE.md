# HIBPBinCreator - Claude Code Instructions

## Project
Downloads HIBP NTLM password hashes and compresses them to binary format.
Python pypsirepacker is the default converter; legacy C++ PsiRepacker.exe is opt-in.

## Conventions
- PowerShell 5.1 compatible -- no `??`, `?.`, ternary, `&&`, or 3-arg `Join-Path`
- All scripts use `[CmdletBinding()]` and comment-based help
- Shared helpers in `lib/HIBPBinCreator.Helpers.ps1` -- dot-sourced by both scripts
- `$ErrorActionPreference = 'Stop'` is set in both main scripts
- pypsirepacker is bundled in `pypsirepacker/` (BSD-3-Clause, from Improsec)

## Key Paths
- Main scripts: `PrepareEnv.ps1`, `BinaryCreator.ps1`
- Helpers: `lib/HIBPBinCreator.Helpers.ps1`
- Python repacker: `pypsirepacker/` (`repacker.py`, `__main__.py`, `__init__.py`)
- Tests: `tests/` (Pester + pytest)
- Docs: `docs/` (Home, Architecture, Operations, Troubleshooting)
- Fixtures: `tests/fixtures/` (sample-ntlm-hashes.txt, sample-ntlm-hashes.bin)

## Testing

### Python tests
```bash
python -m pytest tests/test_repacker.py -v
```

### Pester tests
```powershell
Invoke-Pester tests/ -Output Detailed
```

### Individual test suites
```powershell
Invoke-Pester tests/HIBPBinCreator.Unit.Tests.ps1 -Output Detailed
Invoke-Pester tests/HIBPBinCreator.Behavioral.Tests.ps1 -Output Detailed
Invoke-Pester tests/HIBPBinCreator.Docs.Tests.ps1 -Output Detailed
```

## Config
`config.psd1` is auto-generated by PrepareEnv.ps1 and git-ignored.
Key fields: `RepackerMode` (Python/Legacy), `PythonExe`, `PyPsiRepackerDir`, `PsiRepackerExe`.
